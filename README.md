# Customer Feedback Sentiment Analysis

## Overview

Customer Feedback Sentiment Analysis is a Data Analytics project designed to analyze customer reviews and classify them into Positive, Negative, and Neutral sentiments. The project utilizes Python libraries for data processing, sentiment categorization, and visualization. Various charts are generated to understand customer opinions, satisfaction levels, and overall feedback trends.

## Objective

To analyze customer feedback data and identify sentiment patterns using data analytics and visualization techniques.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The dataset contains customer reviews and sentiment categories:

- Review Text
- Positive Sentiment
- Negative Sentiment
- Neutral Sentiment

## Visualizations

1. Sentiment Distribution (Bar Chart)
2. Sentiment Percentage (Pie Chart)
3. Sentiment Count Analysis (Count Plot)
4. Customer Feedback Sentiment Analysis (Horizontal Bar Chart)

## Conclusion

The Customer Feedback Sentiment Analysis project successfully analyzed customer reviews and categorized them into different sentiment groups. The generated visualizations helped in understanding customer satisfaction levels and identifying feedback trends.

## Outcome

The project effectively demonstrated sentiment analysis using customer feedback data. The visualizations provided meaningful insights into customer opinions, helping businesses understand customer satisfaction and improve decision-making.
